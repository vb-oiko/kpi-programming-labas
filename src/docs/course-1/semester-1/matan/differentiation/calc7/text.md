$$y=\ln (x+\sqrt{a^2 + x^2})

### Розв'язок

$$dy=d\ln (x+\sqrt{a^2 + x^2})
=$$

$$dy=\frac{
    d(x+\sqrt{a^2 + x^2})
}{
    x+\sqrt{a^2 + x^2}
}
=$$

$$dy=\frac{
    dx+d\sqrt{a^2 + x^2}
}{
    x+\sqrt{a^2 + x^2}
}
=$$

$$dy=\frac{
    dx + \frac{
        d(a^2 + x^2)
    }{
        2\sqrt{a^2 + x^2}
    }
}{
    x+\sqrt{a^2 + x^2}
}
=$$

$$dy=\frac{
    dx + \frac{
        xdx
    }{
        \sqrt{a^2 + x^2}
    }
}{
    x+\sqrt{a^2 + x^2}
}
=$$

$$dy=\frac{
    1 + \frac{
        x
    }{
        \sqrt{a^2 + x^2}
    }
}{
    x+\sqrt{a^2 + x^2}
}dx
=$$

$$dy=\frac{
   \frac{
        x +\sqrt{a^2 + x^2}
    }{
        \sqrt{a^2 + x^2}
    }
}{
    x+\sqrt{a^2 + x^2}
}dx
=$$

$$dy=\frac{
    1
}{
